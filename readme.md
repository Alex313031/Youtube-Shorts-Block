# Youtube-short block
Youtubeのショート動画を普通の動画として表示します。

## 処理
仕組みはシンプルで、  
`youtube.com/shorts/"video_id"`  
から、  
`youtube.com/watch?v="video_id"`  
へ自動的にURLを書き換えます。

## よくある質問
